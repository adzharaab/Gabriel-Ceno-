June – March Learning Journey

June – URL / World Wide Web

What I Learned: I learned how URLs work and how the World Wide Web connects different websites on the internet. I also learned that websites are stored on servers and accessed using browsers.

Challenge: At first it was confusing to understand how servers and browsers communicate to display websites.

July – Internet Basics

What I Learned: I learned the basic parts of the internet such as browsers, search engines, and web pages. I also learned how people use the internet to access information.

Challenge: Some internet terms were difficult to remember and understand.

August – HTML Tags & Links

What I Learned: I learned how to use HTML tags to create a webpage and how to connect pages using links.

Challenge: I sometimes forgot to close HTML tags which caused errors in the page.

September – HTML Tables

What I Learned: I learned how to create tables using rows and columns to organize information on a webpage.

Challenge: Aligning the table structure correctly was a bit difficult at first.

October – Login Box & Input Type

What I Learned: I learned how to create input fields such as text boxes, password fields, and buttons.

Challenge: Understanding the different input types and how they work.

November – Website Planning

What I Learned: I learned that planning a website layout helps organize content and makes the design easier.

Challenge: Thinking of a good structure and layout for the website.

December – Multimedia

What I Learned: I learned how to add images, videos, and audio to make webpages more interesting.

Challenge: Adjusting the size of media so it fits well on the webpage.

January – Form Elements

What I Learned: I learned how forms collect user information using text fields, checkboxes, and buttons.

Challenge: Organizing the form so it looks clean and easy to use.

February – CSS Basics

What I Learned: I learned how CSS is used to style webpages by changing colors, layout, and spacing.

Challenge: Remembering the different CSS properties.

March – CSS Font & Color

What I Learned: I learned how to style text using different fonts and colors to make the website look more attractive.

Challenge: Choosing the right color combination and font style.
